# SEN204-Final-Project
Final project submission by Fidan Alasgarli
###
Project Title:
###
Quick Notes App
###
Research Question / Problem Statement:
###
How can users efficiently capture, view, and organize daily personal notes in realtime through a lightweight, accessible web interface without the complexity of
heavy database administration?
###
Objective of the Project:
###
The project aims to design and implement a web application using the Django framework that allows users to seamlessly add, display, and delete notes. The
system relies on a relational database backend and an intuitive user interface to process data dynamically.
